# Python Multi-App Terminal Hub

Welcome to my Python Terminal Hub project! This is a completely console-based application written in Python that acts as a central hub for three distinct mini-programs. 

The project focuses heavily on **modular programming**, using independent functions (`def`) to manage variable scopes and create clean, readable code.

## 🚀 Features

### 1. 🧮 Advanced Calculator
* Supports core mathematical operations: Addition, Subtraction, Multiplication, and Division.
* Features built-in safety logic to prevent `ZeroDivisionError` crashes.
* Fully self-contained execution blocks.

### 2. 🏦 Bank Simulation Portal
* Simulates real-time banking operations.
* Track your current balance, make secure deposits, and execute withdrawals.
* Includes logic checks to prevent users from withdrawing more money than their current balance.
* Utilizes global state management to keep account data consistent.

### 3. 🎮 Rock, Paper, Scissors Game
* Play against an AI opponent utilizing automated random choices.
* Implements fully chained conditional logic (`if-elif-else`) to evaluate complex win/loss matrices efficiently.
* Features an active arcade score tracker keeping tally of both player and computer scores across sequential rounds.

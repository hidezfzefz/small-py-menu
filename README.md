A central menu system that connects different tools together in one executable script.

* **Bank Simulation Module:** Tracks a user's wallet balance with options to check, deposit, or withdraw funds using global state memory.
* **Rock, Paper, Scissors Game:** A complete text-based game against a random AI opponent that tracks sequential scores.

### 2. 🎒 RPG Inventory & Chest Simulator (`Inventory.py`)
A mini-game element that demonstrates dynamic data manipulation.
* Open a mystery chest to randomly pull legendary fantasy loot and add it to your bag.
* View your current items with an active count tracker.
* Safely discard individual items from specific positions inside a Python list.
